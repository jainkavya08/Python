
**Syntax :**
``` python

if condition :
	....
else :
	...
```

```python

age = int(input("Enter your age: "))

if age >= 10 :
	print ("You are eligible to vote")
else:
	print ("You are not eligible to vote")
	
```

### If - Else Ladder Statements
---
```python

marks = int(input("Enter your marks: "))

if marks >= 90 :
	print ("Grade A")
elif marks >= 80 :
	print ("Grade B" )
elif marks >= 70 :
	print ("Grade C")
elif marks >= 50 :
	print ("Grade D")
else :
	print ("Fail")

```
